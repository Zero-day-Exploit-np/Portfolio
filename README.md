# Portfolio

A modern, feature-rich portfolio website with all the latest updates and improvements!

## What's New in This Version

###  **All Features Included & Pre-Integrated:**

 **Smooth Cursor** - Butter-smooth 60fps cursor with GPU acceleration  
 **Lazy Loading** - Images load only when visible for faster performance  
 **Project Filters** - Click buttons to filter by technology (React, Node.js, etc.)  
 **GitHub Stats** - Live contribution graph, top languages, and streak  
 **Full Accessibility** - Keyboard navigation, screen reader support, ARIA labels  
 **PWA Support** - Installable as app, works offline, appears in app drawer  
 **Sound Effects Ready** - Framework in place (just add sound files)  
 **Resume Download** - Button to download your CV  
 **Light/Dark Theme** - Toggle between themes (fully working!)
