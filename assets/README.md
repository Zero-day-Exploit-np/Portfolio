# Assets Folder

Place your images, fonts, and other assets here.

## Recommended Images:
- favicon.ico (16x16 or 32x32)
- og-image.jpg (1200x630 for social media sharing)
- profile-photo.jpg (optional)
- project-screenshots/ (folder for project images)

## Folder Structure:
```
assets/
├── images/
│   ├── projects/
│   │   ├── project1.jpg
│   │   ├── project2.jpg
│   │   └── ...
│   ├── favicon.ico
│   └── og-image.jpg
├── fonts/ (if using custom fonts)
└── icons/ (if using custom icons)
```
